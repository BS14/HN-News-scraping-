# HN-News-scraping-
Automatically scrap the headlines from hackernews and email the content 

# Steps
- Get HN website Front Page
- Scrape Required Content (Title/Link)
- Build Email Body/Content
- Email Authentication 
- Email sent 

# Packages required
- requests
- bs4
- smtplib (default)
- email.mine (default)
- datetime (default)
